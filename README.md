# Amino-Sphere-Illustrator

**A web-based tool to generate publication-ready spherical amino acid sequence diagrams.**

---

## 📌 Overview

AminoSphereIllustrator is a dedicated web visualization tool designed specifically for drawing beautiful, standardized, and publication-ready amino acid sequence schematic diagrams. Each amino acid residue is represented as an independent spherical node, making protein primary structure diagrams clearer, more intuitive, and suitable for papers, presentations, course reports, and academic posters. All rendering and editing run entirely inside the browser. No data upload, no server required, fully private and lightweight.

---

## ✨ Core Features

- **Spherical Residue Visualization** – Each amino acid displayed as an independent spherical node
- **Pseudo-3D Rendering Effect** – Built-in shadow and light simulation for stereoscopic visual effects
- **Chain Rotation Interaction** – Click to rotate subsequent residues; left-click for clockwise, right-click for counterclockwise
- **Adjustable Rotation Angle** – Customize rotation angle and reset with additional click
- **Mouse Wheel Zoom** – Scale canvas freely for detailed viewing
- **Full Visual Style Customization** – Sphere size, color, border thickness, border color
- **Complete Text Control** – Font family, font size, font color for all amino acid labels
- **Multi-Protein Chain Support** – Add and display multiple sequences on one canvas
- **Specific Residue Marking** – Highlight, color-tag, and mark mutation sites and key residues
- **Position Number Display** – Show residue numbers at specified positions
- **Terminal Markers** – Automatically display N-terminus and C-terminus markers
- **High-Resolution Export** – One-click export to editable PowerPoint format
- **Client-Side Only** – No installation, no data transmission, safe and fast

---

## 🎯 Application Scenarios

- Protein primary structure schematic drawing
- Amino acid mutation and functional site marking
- Thesis figure and academic illustration production

---

## 🚀 How to Use

1. **Launch** – Open in any modern browser; no installation required
2. **Input Sequence** – Enter polypeptide sequence, then click "Render Sequence"
3. **Rotate Chain** – Left-click a sphere to rotate clockwise; right-click for counterclockwise; click again to reset
4. **Zoom Canvas** – Use mouse wheel to zoom in and out
5. **Customize Style** – Adjust sphere size, color, border, font family, font size, and text color
6. **Save Colors** – Save preferred colors to palette for reuse
7. **Mark Residues** – Highlight specific positions (e.g., 5-8,12) with custom colors
8. **Show Numbers** – Display residue numbers at specified positions
9. **View Terminals** – Automatic N-terminus (H₂N/NH₂) and C-terminus (COOH/HOOC) markers
10. **Export** – One-click export to editable PowerPoint file

# Amino-Sphere-Illustrator

**一款用于生成出版级别球形氨基酸序列示意图的 Web 工具。**

---

## 📌 概述

AminoSphereIllustrator 是一款专业的 Web 可视化工具，专门用于绘制美观、规范、适合出版的氨基酸序列示意图。每个氨基酸残基以独立球形节点展示，使蛋白质一级结构图更清晰、直观，适用于论文、演示、课程报告和学术海报。所有渲染和编辑均在浏览器端完成，无需上传数据，无需服务器，完全私密且轻量。

---

## ✨ 核心功能

- **球形残基可视化** – 每个氨基酸以独立球形节点展示
- **拟3D渲染效果** – 内置阴影和光照模拟，呈现立体视觉效果
- **链式旋转交互** – 点击小球旋转其后所有氨基酸；左键顺时针，右键逆时针
- **可调旋转角度** – 自定义旋转角度，再次点击可恢复原始方向
- **鼠标滚轮缩放** – 自由缩放画布，便于细节查看
- **全面样式自定义** – 可调节小球大小、颜色、边框粗细、边框颜色
- **完整文字控制** – 支持字体、字号、字体颜色自定义
- **多肽链支持** – 在一个画布中添加和展示多条序列
- **特定残基标记** – 高亮、着色标记突变位点和关键残基
- **位置编号显示** – 在指定位置显示氨基酸序号
- **末端标记** – 自动显示 N 端和 C 端标记
- **高清导出** – 一键导出可编辑的 PowerPoint 文件
- **纯客户端运行** – 无需安装，无数据传输，安全快速

## 🚀 使用说明

1. **启动** – 使用任意现代浏览器打开，无需安装
2. **输入序列** – 输入多肽序列，点击「渲染序列」
3. **旋转链条** – 左键小球顺时针旋转，右键逆时针旋转；再次点击恢复
4. **缩放画布** – 使用鼠标滚轮缩放
5. **自定义样式** – 调整小球大小、颜色、边框、字体、字号、文字颜色
6. **保存颜色** – 将常用颜色保存到调色板以便复用
7. **标记残基** – 输入位置（如 5-8,12）高亮指定残基
8. **显示编号** – 在指定位置显示氨基酸序号
9. **末端标记** – 自动显示 N 端和 C 端标记
10. **导出文件** – 一键导出可编辑的 PowerPoint 文件
