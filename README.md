# Amino-Sphere-Illustrator
A web-based tool to generate publication-ready spherical amino acid sequence diagrams. 

---
📌 Overview
AminoSphereIllustrator is a dedicated web visualization tool designed specifically for drawing beautiful, standardized and publication-ready amino acid sequence schematic diagrams. This tool uses spherical nodes to visually represent each amino acid residue, making protein primary structure diagrams clearer, more intuitive, and more suitable for papers, presentations, course reports and academic posters.
All rendering and editing runs completely inside the browser. No data upload, no server required, fully private and lightweight.

---
✨ Core Features
- Spherical Residue Visualization
Each amino acid is displayed as an independent spherical node, presenting the primary structure of polypeptide and protein in a novel and intuitive way.
- Pseudo-3D Rendering Effect
Built-in shadow, highlight and light simulation to produce real stereoscopic pseudo-3D visual effects, making the diagram professional and high-end.
- Multi-Protein Chain Addition Support
Allows adding and displaying multiple protein or polypeptide sequences in one canvas for comparative drawing and structural arrangement.
- Full Visual Style Customization
Adjustable sphere size, sphere color, border thickness, border color, connecting line thickness and style.
- Complete Text & Font Control
Support custom font selection, font size adjustment and font color modification for all amino acid labels.
- Specific Amino Acid Residue Marking
Highlight, color-tag and mark special amino acids, mutation sites, functional sites and key residues for emphasis and annotation.
- High-Resolution One-Click Export
Export clear high-definition images directly for thesis figures, PPT presentations and academic publishing.
- Client-Side Only, No Installation
Pure front-end web application, no software installation, no data transmission, safe and fast.

---
🎯 Application Scenarios
- Protein primary structure schematic drawing
- Amino acid mutation and functional site marking
- Thesis figure and academic illustration production
- Biochemistry and molecular biology course assignments and reports
- PPT academic presentation and scientific publicity display

---
🚀 How to Use
1. Launch the tool directly in any mainstream modern browser, no local installation or complex environment configuration needed.
2. Input single or multiple polypeptide/protein amino acid sequences into the editing box, supporting standard amino acid notation formats.
3. Tweak core visual parameters freely, including sphere size and color, node border thickness, connecting line weight, to customize the overall diagram style.
4. Adjust label font attributes including font family, font size and font color to match academic display and publishing specifications.
5. Select and mark specific key amino acid residues, mutation sites and functional sites with exclusive colors for prominent visual distinction.
6. Add multiple protein chains on demand, arrange and display different sequence structures on one canvas for comparative visualization.
7. Preview the real-time pseudo-3D rendering effect, fine-tune details until the diagram meets your academic needs.
8. One-click to export high-resolution transparent or solid-background images, directly applicable to papers, PPT presentations and academic reports.
